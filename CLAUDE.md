# CLAUDE.md - 작업 지침서

## 기본 지침

### 1. 대화 언어
- 모든 대화는 한글로 진행한다.
- 기술적 용어는 영어를 그대로 표현하고 코드 관련 설명은 한글로 제공한다.

### 2. 문서 기반 작업 원칙
- 문서를 기반으로 작업하도록 지시받은 경우, 문서의 내용을 반드시 준수한다.
- 문서 내용과 다른 방식으로 작업하지 않는다.
- 문서대로 진행이 어려운 경우에는:
  - 구체적인 문제점과 원인을 리포팅한다
  - 작업을 멈추고 사용자에게 문의한다
  - 사용자의 추가 지시를 받을 때까지 대기한다

### 3. 작업 품질 기준
- **임시방편 금지**: 일시적인 해결책이나 workaround 사용 금지
- **우회방법 금지**: 정해진 절차를 건너뛰거나 다른 방법으로 우회 금지  
- **임의 간소화 금지**: 사용자 허가 없이 작업을 단순화하거나 생략 금지
- 모든 작업은 완전하고 정확하게 수행해야 한다

## 작업 프로세스

1. **요구사항 분석**: 사용자 지시사항과 관련 문서를 정확히 파악
2. **문서 검토**: 해당 작업과 관련된 모든 문서 내용 확인
3. **실행 계획**: 문서 기준에 맞는 구체적인 작업 계획 수립
4. **단계별 실행**: 문서 내용을 정확히 따라 순차적 실행
5. **검증**: 작업 결과가 문서 기준에 부합하는지 확인
6. **문제 발생 시**: 즉시 중단하고 사용자에게 상황 보고
7. **체크리스트**: 문서에 따른 작업을 한 경우, 문서에 체크리스트를 반드시 업데이트
8. **커밋**: 작업 후 반드시 커밋하며, 푸시는 사용자의 지시가 있을때만 진행
   - 커밋 메시지에는 🤖 Generated with [Claude Code] 또는 Co-Authored-By: Claude 등의 서명을 추가하지 않는다
   - 간결하고 명확한 커밋 메시지만 작성한다

## 주의사항

- 시간 절약을 이유로 정해진 절차를 건너뛰지 않는다
- 편의를 위해 문서 내용을 임의로 수정하거나 변경하지 않는다  
- 모든 작업은 문서화된 표준과 절차를 엄격히 준수한다
- 불명확한 부분이 있으면 추측하지 말고 명확히 확인한다
